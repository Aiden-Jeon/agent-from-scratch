## Local Setup

1. Copy the environment template file to your own.
    ```bash
    cp env.template notebooks/.env
    ```
2. Give your api key to the env file
    ```bash
    OPENAI_API_KEY=<YOUR-OPENAI-API-KEY>
    HF_TOKEN=<YOUR-HUGGINGFACE-API-KEY>
    ```
